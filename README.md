# c-shell

Минималистичная UNIX-оболочка, написанная на C. Проект создан для изучения системных вызовов, управления процессами и межпроцессного взаимодействия в UNIX.

## Возможности

- Запуск программ с аргументами
- Конвейеры (`:`)
- Перенаправление ввода/вывода (`>`, `>>`, `<`, `2>`, `2>>`)
- Фоновый режим (`&`)
- Условное выполнение (`&&`, `||`)
- Поддержка всех операторов в любых комбинациях

## Экранирование операторов! (ОСОБОЕ ВНИМАНИЕ)

Shell (bash, zsh) перехватывает специальные символы (`>`, `<`, `&`, `|`, `&&`, `||`) до того, как они попадут в программу. Поэтому перед каждым оператором нужно ставить обратный слеш `\`:

```bash
# Вместо
./program ls -la > out.txt

# Нужно писать
./program ls -la \> out.txt
```

## Примеры использования

```bash
# Простая команда
./program ls -la

# Конвейер
./program ls -la : grep "\\." : wc -l

# Перенаправление вывода
./program echo "hello" \> file.txt

# Дозапись в файл
./program echo "world" \>\> file.txt

# Перенаправление ошибок
./program ls /nonexistent 2\> error.log

# Фоновый режим
./program sleep 10 \&

# Условное выполнение
./program ls /tmp && echo "ok"
./program ls /nonexistent || echo "not found"

# Всё вместе
./program cat \< in.txt : grep hello : wc -l \> count.txt && echo "done" \&
```

```bash

```

Учебный проект для понимания:
- Системных вызовов (`fork`, `exec`, `pipe`, `dup2`, `wait`)
- Сигналов и управления процессами
- Перенаправления ввода/вывода
- Межпроцессного взаимодействия (IPC)
